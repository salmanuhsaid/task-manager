About TaskFlow

Learn what this app does and how it helps you

🎯 Purpose of the App

TaskFlow is a browser-based task management application designed to help students and professionals organize their daily tasks efficiently. It allows users to create, prioritize, track, and delete tasks — all without needing an internet connection or a server.

🛠️ Technologies Used

  • HTML5 — Structure and layout of all pages
  • CSS3 — Styling, colors, animations, and responsive design
  • JavaScript — All dynamic features and interactions
  • localStorage — Browser-based data storage (database layer)

⚙️ Key Features

  • Add tasks with a title, due date, and priority level
  • Mark tasks as complete or undo completion
  • Delete tasks you no longer need
  • Filter tasks by All, Active, or Completed
  • All data is saved automatically in your browser

💾 How Data is Stored

TaskFlow uses the browser's built-in localStorage system to save your tasks. This means your tasks are stored directly on your device and will still be there even after you close and reopen the browser. No server or external database is needed.
